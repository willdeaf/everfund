== Description ==

### Integrate donation experiences, faster.

Everfund is the dev-first donation platform for modern nonprofits. Integrate donation experiences with a composable SDK and all the components modern nonprofits need, without building from scratch.

Use our composable SDK to take control of all the components needed for donations in modern nonprofit websites, without building from scratch. With our tools, you'll save time learning the complexities of payment systems and integrate donation experiences in the shape mdoern nonprofits need them so they can focus more on their causes.

### Features for high performance donations

Everfund manages everything you need to take a donation, not just a payment. Make nonprofit WordPress websites into high performance fundraising machines with all the features they need to take full donations from modern donors:

1. Single Donations
2. Recurring Donations
3. Donation
4. GDPR Consent
5. Cover Processor Fees
6. Donation Emails
7. Customisable donation checkouts
8. Apple / Google Pay

### Secure, low maintenance and always up to date

Everfund keeps everything secure and updated without having to worry about updating plugins when changes are made. It helps nonprofits to focus on their fundraising and leaves you less liable for ongoing maintenance.

== Installation ==

Upload the plugin files to the `/wp-content/plugins/everfund-sdk` directory, or install the plugin through the WordPress plugins screen directly.
Once uploaded using the editor add the everfund code to a button or add html code? 

Stuck? Check out our developer documentation [here](https://docs.everfund.com/docs/js-sdk/wordpress)
